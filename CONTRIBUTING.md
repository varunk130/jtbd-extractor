# Contributing to JTBD Extractor

Thanks for your interest! Here's how to contribute:

## Quick Start
1. Fork and clone the repo
2. Create a branch: `git checkout -b my-feature`
3. Make your changes and commit with clear messages
4. Push and open a Pull Request

## What We're Looking For
- New example outputs for different industries
- Improvements to the HTML visual report template
- Additional job categories beyond Functional/Emotional/Social
- Better opportunity scoring algorithms

## Code Style
- Python: Follow PEP 8
- Markdown: Use consistent header levels
